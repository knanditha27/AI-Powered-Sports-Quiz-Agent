# 🏆 AI-Powered Sports Quiz Generation Agent
An AI-powered application that generates sports quiz questions based on the selected sport and difficulty level.

## 🚀 Live Demo
https://ai-powered-sports-quiz-agent.streamlit.app

## ✨ Features
- Select different sports
- Choose difficulty levels
- Generate AI-powered quiz questions
- Simple and interactive Streamlit interface

## 🛠️ Technologies Used
- Python
- Streamlit
- Google GenAI
- ChromaDB
- Sentence Transformers

## ▶️ Run Locally
1. Clone the repository
2. Install the dependencies:
   `pip install -r requirements.txt`
3. Run the application:
   `streamlit run app.py`
